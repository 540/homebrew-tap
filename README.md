# homebrew-tap

Homebrew tap de [540](https://github.com/540) para distribuir herramientas internas.

## Scribe

Notas de reunión bot-free, transcripción 100 % local — [540/Scribe](https://github.com/540/Scribe).

```sh
brew tap 540/tap
brew install --cask scribe
```

Actualizar:

```sh
brew upgrade --cask scribe
```

> El código fuente de Scribe es privado; aquí solo se publica el binario empaquetado.
